# 🎉 AI Birthday Wish Generator

Create personalized birthday wishes instantly using the power of AI! 🤖🎂

---

## ✨ Features

✅ Generate unique birthday wishes in seconds  
✅ Multiple tones (Funny, Formal, Emotional, Romantic, Friendly)  
✅ Easy-to-use Streamlit interface  
✅ AI-powered using Groq LLM  
✅ Fast and lightweight

---

## 🛠️ Tech Stack

- 🐍 Python
- 🎨 Streamlit
- 🤖 Groq API
- 🔑 Environment Variables (.env)

## 🚀 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/AI_Birthday_Wish_Generator.git
cd AI_Birthday_Wish_Generator
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Add API Key

Create a `.env` file.

```
GROQ_API_KEY=your_api_key_here
```

### 4️⃣ Run the App

```bash
streamlit run app.py
```

## 🎯 How It Works

1. Enter the person's name.
2. Choose the relationship.
3. Select the desired tone.
4. Click **Generate Wish**.
5. Get an AI-generated birthday message instantly.
